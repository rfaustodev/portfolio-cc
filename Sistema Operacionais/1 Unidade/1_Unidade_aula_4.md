# Composição dos Sistemas Operacionais

A composição típica de um sistema operacional se refere á estrutura e aos principais componentes que compõem um S.O padrão. São eles responsavel por lidar com gerenciamento de servicos, recuros em coneção com as pedidos dos usúarios ao computador.

Os principais componentes da arquitetura de um sistema operacional incluem o núcleo(kernel), utilitários, sistema de arquivos, drivers de dispositivos, interfaces com o usúariose assim por diante, cada um deles desempenha um papel fundamental ao sistema operacional.

Os componentes são inter-ligados para trabalha juntos com função de fornecer funcionalidades essenciais para o computador, aqui estão as principais:

- **Núcleo(kernel):** O nucleo é a parte central do sistema operacional e é responsável por gerenciar recursos do hardware, como CPU, Memória, dispositivos E/S e gerenciamento de processos.

- **Drivers de dispositivo:** Os drivers de dispositivos são programas ou módulos de permitem que o sistema operacional se cominuque com dispositivos de hardware específicos, como placas de vídeo, impressoras, discos rígidos etc, Eles traduzem comandos do sistema operacional em instruções compreensíveis pelo hardware.

- **Shell:** O shell é a interface que permite que os usuários interagem com o sistema operacional, Pode ser uma iinterface de linha de comando(CLI) ou uma interface gráfica de usúario(GUI), O shell interpreta comandos ou ações do usuarios e os encaminha para o núcleo para execução.

- **Biblioteca de Sistema:** As bibliotecas de sistema são coleção de funções que são usados por aplicativo para realizar tarefas comun, como entrada e saida, manipilação de arquivos e comunicação com sistema operacional, Isso permite que os desenvolvedores de reutilizem código e simplifique o desenvolvimento de aplicativos.

E outros componentes que pode incluir em um sistema operacional:

- **Utilítarios do sistema:** Os utilítarios são programas que auxiliam na administração e manutenção do sistema. Isso incluir utilítarios de gerenciamento de disco, utilitários de diagóstico, ferramentas de segurança, entre outros.

- **Aplicativos:** São programas que realizam tarefas especificas. Isso incluir aplicativos de estudos, jogos, navegadores e entre outros.

- **Sstema de arquivos:** O sistema de arquivos é responsável por organiza, armazenar e recuperar dados em dispositivos de armazenamentos, como discos rígidos e SSDS, Ele determina a strutura dos diretórios, a forma como arquivos são nomeados e como os dados são organizados no armazenamentos.

- **Serviços de rede:** Em sistema operacional de rede, há componentes para gerenciar a conectividade de rede, incluir configurações de rede, protocolos de comunicação, serviços de compartilhamento de arquivos e serviços web.

Esses componetes trabalham juntos para fonercer o ambiente para o usuario final.
